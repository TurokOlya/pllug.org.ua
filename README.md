#### Runing ui theme locally

- install nodejs from [Official site](https://nodejs.org/en/)

- installing gulp and running service

```bash

npm install -g gulp

cd ui-theme

npm i

gulp

```

App will run on port [http://localhost:3000](http://localhost:3000)
